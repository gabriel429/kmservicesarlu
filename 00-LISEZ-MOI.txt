╔════════════════════════════════════════════════════════════════════════════╗
║                      DÉMARRAGE RAPIDE (5 MIN)                             ║
╚════════════════════════════════════════════════════════════════════════════╝

Étape 1: CRÉER LA BASE DE DONNÉES
─────────────────────────────────
1. Ouvrez phpMyAdmin: http://localhost/phpmyadmin
2. Créez une base "km_services"
3. Importez le fichier: database/schema.sql
4. C'est prêt!

Étape 2: CONFIGURER L'APP
─────────────────────────
Ouvrez config/config.php et vérifiez:
• DB_HOST: localhost
• DB_USER: root
• DB_PASS: (votre mot de passe)
• DB_NAME: km_services

Étape 3: VÉRIFIER L'INSTALLATION
────────────────────────────────
1. Lancez WAMP/XAMPP
2. Ouvrez: http://localhost/kmservices/check.php
3. Tout doit être ✅

Étape 4: LANCER LE SITE
──────────────────────
• Public: http://localhost/kmservices/
• Admin: http://localhost/kmservices/admin/login

Étape 5: SE CONNECTER
────────────────────
Utilisateur: admin
Mot de passe: password
⚠️ À CHANGER IMMÉDIATEMENT!

Étape 6: COMMENCER
──────────────────
1. Allez dans "Projets"
2. Créez votre premier projet
3. Ajoutez des produits
4. Personnalisez le contenu
