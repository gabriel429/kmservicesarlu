📊 Points de Rupture (Breakpoints)

Desktop (≥1024px)
@media (min-width: 1025px)
• Sidebar fixe 250px gauche
• Layout 2 colonnes (250px + 1fr)
• Menu toggle: display:none
• Toutes colonnes visibles
• Fonts normales
Tablet (768px - 1024px)
@media (max-width: 1024px)
• Sidebar drawer visible au click
• Max-height: 70vh
• Menu toggle: display:block
• Header padding réduit
• Font-size tables: 0.85rem
Mobile (480px - 768px)
@media (max-width: 768px)
• Sidebar drawer: max-height 65vh
• Position: fixed bottom
• Tables format "carte"
• Boutons 100% width
• Font-size 16px inputs (iOS)
• Main-content margin-bottom 300px
Petit Mobile (<480px)
@media (max-width: 480px)
• Tables ultra-compactées (0.65rem)
• Cellules: padding-left 40%
• Sidebar très réduit
• Font sizes minimales
• Espacements réduits

👨‍💻 Checklist pour Développeurs

  • Sidebar transformée en drawer mobile
  • Bouton toggle menu (hamburger) ajouté
  • JavaScript pour gérer ouverture/fermeture du menu
  • Tous les fichiers admin ont data-label sur cellules
  • CSS media queries pour 4 breakpoints
  • Tables format "carte" sur mobile (via CSS ::before)
  • Formulaires 100% width sur mobile
  • Font-size 16px inputs pour éviter zoom iOS
  • Boutons pleine largeur sur mobile
  • Animations icône menu (fa-bars ↔ fa-times)
  • Menu ferme au click externe / Escape
  • Menu ferme au click sur lien
  • Page active marquée dans menu
  • CSS variables utilisées pour thèmes
  • Scroll horizontal pour tables si besoin
